# RCT Ecosystem – December 2025 Simulated Benchmark (Public Pack)

This folder is the **public benchmark pack** for the RCT Ecosystem.

It is designed to be published as a standalone GitHub repository,
for example as:

```text
rct-ecosystem-benchmark-public/
```

## Contents

- `README.md` – this file
- `RCT-Ecosystem-December-2025-Simulated-Benchmark-Report.pdf`  
  Full simulated benchmark report (December 2025)
- `RCT-Executive-Summary-Simulated-Benchmark_PUBLIC.md`  
  Executive summary for non‑technical readers
- `00-MASTER-INDEX-Benchmark-Public.md`  
  Index of all benchmark/experiment documents
- `RCT-December-2025-Benchmark-Plan.md`  
  High‑level plan for the December 2025 benchmark
- `RCT-Experiment-Scope-Complete.md`  
  Scope definition: what is included / excluded in this benchmark
- `RCT-Comprehensive-Benchmark-Framework.md`  
  Generic framework for running RCT benchmarks in future rounds
- `RCT-Comprehensive-Research-Assessment.md`  
  Research‑oriented assessment and positioning
- `RCT-Experiment-Explanation-Complete.md`  
  Human‑readable explanation of the experiment phases and methodology

> **Note:** All numbers in this pack come from a
> **simulated benchmark**. They are meant as engineering
> signals and research artefacts, **not** as production SLAs or
> formal marketing claims.

## Limitations & disclaimers

- All results are **simulated** and generated by the RCT team.
- No external lab or third‑party has audited or reproduced
  these numbers yet.
- Real‑world performance may differ depending on hardware,
  model configuration, knowledge base size, and traffic pattern.

## How to use this pack

- For **architects / engineers** – treat this as a detailed
  reference for how RCT thinks about benchmarking and
  production readiness.
- For **researchers** – you can reuse the methodology,
  phases and metrics as a template for your own systems.
- For **executives / partners** – focus on the Executive
  Summary and the high‑level conclusions, keeping the
  simulated nature of the benchmark in mind.
